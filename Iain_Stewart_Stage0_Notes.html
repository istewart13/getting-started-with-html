How The Web Works

The world-wide web (www) is a collection of computers that communicate with each other over a very wide area. The internet uses a protocol called HTTP to communicate with you, your browser and servers. When a user enters a website their computer sends an HTTP request to a server (computers that host the files that make up the web), which then finds and returns the relevant files. The user’s web browser then uses the returned files to display content to the user.

HTML

HTML stands for HyperText Markup Language. HTML documents make up most of the content on the web. They can contain text content, markup, references to other documents (images, videos etc) and links to other pages. HTML documents are made up of HTML elements, this refers to everything within a set of opening and closing tags.

HTML Tags

Markup consists of tags, which can also be nested. Some tags (called "void" tags) do not require a closing tag.

Computers Are Stupid

They cannot think for themselves and need to be told what to do very precisely by the programmer. The interpret instructions very literally and so simple errors, such as a missing “>” in a tag, can cause a program not to run correctly. Programmers therefore need to write exactly the way a computer understands (also known as writing with correct "syntax").

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.
